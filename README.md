# DevOps & Automation

This repository serves as a centralized hub for my DevOps engineering projects, automation scripts, and infrastructure configurations. It documents my expertise in cloud-native technologies, CI/CD pipelines, and systems administration.

## 📂 Repository Structure

The repository is organized into specific domains of the DevOps lifecycle:

* **Ansible**: Playbooks and roles for automated configuration management and server orchestration.
* **Kubernetes deployments**: YAML manifests for deploying scalable applications, including Deployments, Services, and Ingress configurations.
* **Shell**: Bash scripts designed for system automation, monitoring, and routine maintenance tasks.
* **Groovy**: Jenkins pipeline scripts (Jenkinsfiles) and shared library components.
* **Python**: Scripts focused on Kubernetes operational tasks and cluster stress testing.
* **HTML**: Sample web projects used for testing deployment pipelines and web server configurations.

## 🛠️ Tech Stack & Tools

* **Orchestration:** Kubernetes
* **Configuration Management:** Ansible
* **CI/CD:** Jenkins (Groovy)
* **Scripting:** Bash, Python
* **Version Control:** Git
